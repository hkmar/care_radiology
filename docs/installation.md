# Installation

## Stable release

To install Care Radiology Plugin, run this command in your terminal:

```sh
pip install Care Radiology Plugin
```

## From source

The source files for Care Radiology Plugin can be downloaded from the [Github repo](https://github.com/ohcnetwork/care_radiology).

You can either clone the public repository:

```sh
git clone git://github.com/ohcnetwork/care_radiology
```

Or download the [tarball](https://github.com/ohcnetwork/care_radiology/tarball/master):

```sh
curl -OJL https://github.com/ohcnetwork/care_radiology/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd care_radiology
pip install .
```
